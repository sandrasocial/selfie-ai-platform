Add Module 1 — Start Here

📁 Context: We are working inside the existing StarterKitCourse.jsx file, within the STARTER_KIT_COURSE data structure and LessonContent rendering system.
🧱 Objective: Create Module 1 with 1 lesson titled "Your Brand, Your Power", containing video, styled text, download cards, reflection prompt with AI quote generation, and a minimalist checklist.
🧬 Design Standards:

Typography: Cormorant Garamond (headlines), Inter Light (body)

Colors: #171719, #4C4B4B, #F1F1F1

Style: No gradients, no icons/emojis, flat layout, editorial-inspired

Button: uppercase, black bg, white text; hover: inverted

✅ TASKS TO IMPLEMENT:
