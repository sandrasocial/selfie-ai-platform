5. 🧑‍🎨 Brand-Locked Design Notes
Fonts:


Cormorant Garamond (serif): Headlines, quotes


Neue Einstellung (sans): Body, buttons, input fields


Color Palette:


Background: #F1F1F1 (Soft White)


Text: #171719 (Luxury Black), #4C4B4B (Deep Graphite)


Accent: #B5B5B3 (Warm Gray)


Components:


Buttons: Transparent or black border, all-caps, no shadows


Dividers: 1px Warm Gray line, 60% width, center-aligned


Notes section: Light gray text area with serif label above



6. 📱 Mobile-First Requirements
Vertical stack layout for modules


Auto-expand current module/lesson


Sticky footer CTA: “Next Lesson” / “Mark Complete”


Video player adaptive height


One-column content blocks (no sidebars)



7. ✨ Suggested UX Animations (Optional)

Progress bar animates on scroll


Save Note → checkmark + “Saved” appears briefly
