Explanation of Key Changes
Role-play Header
The new prompt begins with “You are ‘Sandra AI,’ speaking exactly as Sandra would….” That forces OpenAI to fully adopt your tone without slipping into generic motivational language.

Strict “One-Caption” Output
We dropped the old array of 3 captions. Now it must produce exactly one caption object, so you get full length (120–180 characters) and deeper detail every time.

Concise “Problem → Transformation → Call-to-Action” Framework
We enumerated Hook, Before, Shift, The Now, Takeaway, and Call-to-Action. OpenAI will fill each in a single block of text, which you then display under “Hook” and “Caption” in the front end.

Length Check
By specifying “120–180 characters total,” you guarantee it isn’t too short or generic. If OpenAI returns fewer characters, you can surface an error or loop back.

Signature Emojis
Limiting to exactly 1–2 emojis from your list makes every caption feel on-brand.
