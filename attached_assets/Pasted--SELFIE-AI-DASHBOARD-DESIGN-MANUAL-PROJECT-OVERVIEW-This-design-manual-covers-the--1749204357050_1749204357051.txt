## âś¨ VISUAL SYSTEM

**Typography:**

* Headings: `Cormorant Garamond` (serif, elegant, editorial)

**Color Palette:**

* Buttons = uppercase, border only (no icons)
* Hover effects = soft scale + drop shadow
* Background imagery = desaturated
* Offer Cards = Grayscale images with section-specific hierarchy

**Global Components:**

* Header: Already implemented (transparent on hero, solid on scroll)
* Footer: Consistent across all pages (white text, no icons)
