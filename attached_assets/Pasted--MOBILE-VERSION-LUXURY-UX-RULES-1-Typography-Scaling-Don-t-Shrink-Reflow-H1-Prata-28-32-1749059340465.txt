📱 MOBILE VERSION — LUXURY UX RULES
✅ 1. Typography Scaling (Don’t Shrink, Reflow)
H1 (Prata): 28–32px on mobile, still bold and spaced out

Body (Helvetica Neue): 15–16px, no lighter than #4C4B4B

Buttons: 14px uppercase, always centered and tappable (min width: 48px)

👉 Don’t reduce size too much—reduce padding instead so your fonts still feel premium.

🖤 5. Image Grids (Before/After, Template Previews, etc.)
Switch from 3-column to 1 or 2-column stacked layout

Add swipeable carousel for side-scroll content (like presets, templates, etc.)

Keep border spacing and clean gutters—no cramped grids

🎯 6. Call-To-Actions
Always centered and thumb-friendly

CTA buttons should span ~80–90% width

Use icon (📸 or →) only if it enhances clarity, never clutter

🧼 7. Reduce Visual Clutter
Remove icons that don't serve UX clarity

Collapse stats (like 120K followers, $1M goal) into clean horizontal scroll OR stack with dividers
