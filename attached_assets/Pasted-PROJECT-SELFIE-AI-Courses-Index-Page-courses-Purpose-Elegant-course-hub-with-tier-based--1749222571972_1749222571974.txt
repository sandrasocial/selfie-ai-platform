🎓 Course Cards Grid
Layout:
1 row of 3 cards
Responsive: Stack vertically on mobile

Each card shows image, title, subtitle, CTA, and state logic

🧱 Card Components (per course):
Element	Styling
Background Image	Full-cover, editorial photo
Overlay	B&W for locked, normal for unlocked
Course Title	Cormorant Garamond, 24px
Subheadline	Neue Einstellung, 14px, light gray
CTA Button	Transparent, border-only, uppercase — different label based on state
Lock Icon (if locked)	Top-right, minimal line icon
Progress Bar (if unlocked)	Thin black bar at bottom with %

🪄 STATE-BASED VISUAL LOGIC
✅ If User Has Access:
Image: Full color
CTA Button: "Start Course" → links to specific course page
Progress: Show bar or “X% complete”
Cursor: Pointer on hover

🔒 If User Does Not Have Access:
Image: Grayscale or low-opacity overlay (opacity: 0.5)
CTA Button: "Unlock with Plan" → links to sales page
Lock Icon: Top-right corner
Hover: Slight lift/shadow effect still applies
