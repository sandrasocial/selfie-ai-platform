2. Integration Checkpoints
Dashboard Integration:

Check hasProfile on dashboard load
Redirect to onboarding if no profile exists
Show personalized welcome message using profile data

AI Tool Integration:

Every AI call should check for profile using useProfile()
Pass profile context to all prompts
Show "complete profile for better results" when no profile exists

Workbook Generation:

Combine course content + user input + profile data
Send to AI for personalized generation
Use result in PDFMonkey template with profile-specific fields

For Your Platform:
