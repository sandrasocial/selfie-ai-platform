🖼️ INDIVIDUAL DROP CARD
Each card should feel like a magazine cover meets tool kit.

CARD ELEMENTS:

Element	Description
📌 Title	Top left or centered — large serif (e.g., “Visual Confidence Templates”)
🔖 Type Badge	Small uppercase tag: Template, Audio, Mini-Course, AI Tool
📝 Short Desc	1–2 line description in Neue Einstellung
📅 Drip Lock	If availableFrom > today, card appears blurred with lock icon + text: “Coming [Date]”
🔓 Unlocked	CTA: “Open” or “Download” — appear on hover
🔒 Locked	CTA: “Unlock with Plan” (or “Coming Soon”)

Visual Cues:

Locked = blurred image, grayscale card, lock icon at top right

Unlocked = sharp color photo or layout, hover glow
