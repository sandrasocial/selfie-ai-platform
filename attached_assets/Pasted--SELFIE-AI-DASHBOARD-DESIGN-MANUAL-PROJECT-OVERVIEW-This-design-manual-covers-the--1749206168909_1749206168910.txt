**SELFIE AI™ DASHBOARD DESIGN MANUAL**

---

### 📊 PROJECT OVERVIEW

This design manual covers the **Member Dashboard** for SELFIE AI™ — a personalized brand-building HQ inspired by editorial luxury brands like Tom Ford, Vogue, and Chanel. The dashboard blends minimalism, emotional depth, and strategic functionality to guide users through their empowerment journey.

---

## ✨ VISUAL SYSTEM

**Typography:**

* Headings: `Cormorant Garamond` (serif, elegant, editorial)
* Body/UI: `Neue Einstellung` (clean sans-serif)

**Color Palette:**

* Luxury Black: `#171719`
* Soft White: `#F1F1F1`
* Warm Gray: `#B5B5B3`
* Graphite: `#4C4B4B`

**Style Rules:**

* All corners = 0px (sharp)
* Buttons = uppercase, thin border only, thinner font
* Hover effects = soft scale + drop shadow
* Background imagery = NO grayscale if using branded photos; instead apply `rgba(0, 0, 0, 0.3)` overlay
* Offer Cards = Luxury editorial photos with toned styling
* Tool Icons = Thin, transparent white line icons (modern, minimal)

**Global Components:**

* Header: Already implemented (transparent on hero, solid on scroll)
* Footer: Consistent across all pages (white text, no icons)
