Please treat Module 5 as a feature module — not just static content. This module contains the interactive Aesthetic Builder we’ve started building. Here's what needs to happen:

✅ Phase 1: User Selects Vibe → AI Generates Aesthetic
When the user selects a vibe (e.g. "Bold Femme", "Neutral Luxury"):

Frontend sends a POST to /api/ai/visual-strategy with the vibe ID.

Backend returns:

ts
Kopier
{
  images: [array of curated image URLs],
  palette: [array of HEX colors],
  fonts: [Google Font names],
  keywords: [string[]],
  strategy: string
}
Display a layout grid with:

6–9 image cards (styled like an IG grid)

A color palette strip (swatches or tags)

Font names shown in their actual typeface

A short AI-generated visual strategy statement

3–5 brand keywords

⚠️ The curated image links come from our pre-sorted library (use placeholder postimg.cc links for now).

✅ Phase 2: Make It Interactive + Editable
Once the aesthetic is generated:

Allow drag-and-drop (React Beautiful DnD or similar) to rearrange images

Add file upload support to replace images in the grid

Let the user:

Regenerate color palette or captions via AI

Edit keywords or strategy

Preview changes live

Provide two export options:

Download as a PDF brand board (PDFMonkey or pdf-lib)

Export as a grid preview image for social media
