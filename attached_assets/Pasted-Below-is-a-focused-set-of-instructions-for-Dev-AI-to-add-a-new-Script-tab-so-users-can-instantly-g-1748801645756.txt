You are Sandra’s AI Story Coach. Based on the user’s story ({storyInput}) and their chosen tone (e.g., “empowering,” “conversational,” “inspirational”), generate **ONE** short Instagram–friendly video script. This script must include:
1) A brief **voice-over narration** (about 2–3 sentences) that opens with a “Hook” (attention-getter).

Follow the “Becoming HER” framework loosely—Hook → Before (context) → Now (what’s happening) → Takeaway/CTA—all in a single script block.  
Total length: 3–4 short paragraphs. Do NOT mention “Sandra,” only speak as the user’s voice.  
Use 1–2 emojis appropriately (no more).

Only return the final script block—no extra JSON, no markdown, just plain text.

import OpenAI from "openai";
const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

app.post("/api/generate-script", async (req, res) => {
  const { storyInput, tone } = req.body;
  if (!storyInput || storyInput.trim().length < 10) {
    return res.status(400).json({ error: "Please provide at least 10 characters…" });
  }

  const prompt = `
    You are Sandra’s AI Story Coach. Based on the user’s story (“${storyInput}”) and their chosen tone (“${tone}”), produce one short Instagram video script:
    1) Voice-Over (Hook → context → now → takeaway/CTA).  
    2) B-Roll cue indicating what visuals to show.  
    3) On-Camera line guiding the user to speak.  
    Use 1–2 emojis max. Output only the final script block in plain text.
  `;

  try {
    const response = await openai.chat.completions.create({
      model: "gpt-4o-mini", // or whichever high-quality model you’re using
      messages: [{ role: "user", content: prompt }],
      temperature: 0.8,
      max_tokens: 200,
    });
    const script = response.choices?.[0]?.message?.content?.trim() || "";
    return res.json({ script });
  } catch (err) {
    console.error("OpenAI Script Error:", err);
    return res.status(500).json({ error: "AI service error—please try again." });
  }
});
